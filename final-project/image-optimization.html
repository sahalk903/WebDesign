    What does image optimization mean?

    Image optimization reduces the size of your images to its smallest size without ruining the quality of the images and the page load times remain at a minimum. This is completed while also keeping the right format, size, dimension, and resolution. Image optimization improves a website by page load speed and creating the user interface to be useful to create a positive user experience. This can be done is several different ways including resizing the images, caching, or compressing the size.
